Dado a um array de números inteiros e um valor inteiro target, retorne os indices de dois números onde a soma deles é igual ao target.
Você talvez irá assumir que cada input terá apenas uma solução, e você não poderá usar o mesmo elemento duas vezes.
Você pode retornar a resposta em qualquer ordem.

Exemplo 1:
Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].

Examplo 2:
Input: nums = [3,2,4], target = 6
Output: [1,2]

Examplo 3:
Input: nums = [3,3], target = 6
Output: [0,1]
